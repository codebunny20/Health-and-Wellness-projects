TAWT Overview

A lightweight, client‑side web app to track daily wellness, medication/HRT adherence, transition milestones, and cycles. Data is stored locally in your browser (no server).

Tabs and features

Track

  • Log Date, Mood (1–5), Energy (1–5), Sleep (hrs)
  • Optional meds/HRT: Taken/Skipped, Dose, Name, Time
  • Notes for symptoms, triggers, wins
  • Recent Entries list with quick delete

Milestones

  • Add items with Date, Title, Notes
  • Timeline view with Edit and Delete

Cycle

  • Track start date (Day 1), optional end date, flow (light/medium/heavy), and notes
  • Actions: Save Cycle, Reset, Mark Today as Day 1
  • Cycle Stats: Average Length and predicted Next Start
  • Recent Cycles list with quick delete

Reports

  • Averages over a selectable range (7/14/30 days): Mood, Energy, Sleep
  • Medication adherence (% taken among entries that recorded meds)
  • Mood variability (standard deviation)
  • Sparklines for Mood and Energy

Settings

  • Export JSON backup
  • Import JSON to restore
  • Clear All data
  • Data info: counts and approximate size

Data model (localStorage)

Keys

  • tawt_entries: array of daily entries
  • tawt_milestones: array of milestones
  • tawt_cycles: array of cycle records

Entry

  • id, date (YYYY‑MM‑DD), mood, energy, sleep
  • meds: { taken: "yes" | "no" | null, dose, name, time }
  • notes (string)

Milestone

  • id, date, title, note

Cycle

  • id, start (YYYY‑MM‑DD), end (optional YYYY‑MM‑DD), flow ("light" | "medium" | "heavy" | null), notes

How it works

  • Single‑page app; no external libraries.
  • All calculations (KPIs, sparkline rendering) are done in the browser.
  • Dedupes entries by date (latest wins). Most recent items are shown first in lists.
  • Cycle stats: average length is computed from consecutive start dates; next start is predicted from the average.

Privacy and accessibility

  • 100% local: uses browser localStorage only.
  • Keyboard and screen‑reader friendly UI with clear labels and live regions for lists.

Quick start

  1. Open the Track tab and save your first daily entry.
  2. Add milestones for key transition events.
  3. Open Reports to view averages, variability, and trends.
  4. Periodically Export JSON for backup; Import to restore.
  5. Optionally, use Cycle to mark Day 1 today; later add end date, flow, and notes.
